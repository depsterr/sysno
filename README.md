sysno
=====
Shellscripts for viewing syscall numbers

installation
------------
```
make install
```

usage
-----
Get syscall number for a specific syscall
```
sysno SYSCALL [32/64]
```

Browse syscalls in pager
```
sysbrowse 32/64
```
